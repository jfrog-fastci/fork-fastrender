# Pageset page loop (`pageset_page_loop`)

This workstream fixes pages **one-by-one** by turning real failures into minimal, spec-correct capability improvements.

## Owns

- Reproducing a pageset failure deterministically
- Adding a regression (fixture/WPT/unit test)
- Implementing the smallest spec-correct change that fixes the failure

## Does NOT own

- Broad refactors without a pageset-driven outcome
- Browser UI feature work
